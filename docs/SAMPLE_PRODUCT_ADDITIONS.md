### **Step 1: Basic Information**
```json
{
  "title": "Product Title",
  "description": "Detailed Product Description",
  "brand": "Brand Name"
}
```

### **Step 2: Pricing & Region**
```json
{
  "price": "Selling Price (in smallest currency unit)",
  "originalPrice": "Original/MSRP Price",
  "currency": "NGN|GHS|XOF|KES|ZAR|ATH",
  "location": {
    "country": "Country Name",
    "city": "City Name"
  },
  "quantity": "Stock Quantity"
}
```

### **Step 3: Categories & Condition**
```json
{
  "category": "Main Category",
  "subcategory": "Subcategory",
  "condition": "new|used|refurbished"
}
```

### **Step 4: Product Attributes**
```json
{
  "attributes": [
    {
      "key": "Attribute Name (e.g., Color, Size)",
      "value": "Values (comma-separated)"
    }
  ]
}
```

### **Step 5: Product Variants** (Auto-generated from attributes)
```json
{
  "variants": [
    {
      "sku": "Product SKU (optional)",
      "price": "Variant Price Override (optional)",
      "quantity": "Variant Stock",
      "attributes": {
        "Color": "Red",
        "Size": "M"
      }
    }
  ]
}
```

### **Step 6: Shipping Options**
```json
{
  "shippingOptions": [
    {
      "method": "Shipping Method Name",
      "cost": "Shipping Cost",
      "estimatedDays": "Delivery Time in Days"
    }
  ]
}
```

### **Step 7: Additional Settings**
```json
{
  "tags": "Comma-separated tags",
  "discount": "Discount Percentage (0-100)",
  "featured": "Boolean - Featured Product"
}
```
